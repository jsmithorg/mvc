Packageorg.jsmith.mvc.view
Interfacepublic interface IViewRegister extends flash.events.IEventDispatcher
ImplementorsViewRegister

The IViewRegister is an interface for the single access point to all the application's view instances (IView).

IView instances must be registered using the class that implements this interface in order to participate in the MVC framework.

Classes the implement IViewRegister must keep track of and contain references to any IView instances registered using the register(view:IView) method.
