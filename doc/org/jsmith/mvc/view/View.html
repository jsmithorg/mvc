Classpublic class View
InheritanceView flash.display.MovieClip
ImplementsIView

The graphical component in our Model-View-Controller framework.

View provides an abstract class with convenience properties that access the 'Model,' 'View,' 'Controller' and event dispatching objects.


Example
To use the View class, create a concrete view that subclasses the abstract View class: 
  
    package
    {
     import org.jsmith.mvc.view.View;
     import org.jsmith.mvc.view.IView;
     ...
  
     public class MyClassView extends View implements IView
     {
      public function MyClassView()
      {
       ...
      }
  
     }
    }

controllerproperty
protected static var controller:IController

The Controller instance.

In general, you will not need to use this. It's here, however, just in case. Normally, the commands will be invoked using the MVCEventDispatcher which then invokes Controller. Because of this, you must invoke the executeCommand method using an Event object, and the ICommand must be registered in the controller to invoke with said Event.

See also

org.jsmith.mvc.events.MVCEvent
org.jsmith.mvc.controller.command.Command
org.jsmith.mvc.controller.FrontController

Example
To use the Controller instance to execute an ICommand: 
   
     package
     {
      public class MyClassView extends View implements IView
      {
       public function MyClassView()
       {
        ...
       }
   
       public function myMethod():void
       {
        //first the Command must be registered
        controller.registerCommand(Event.EVENT, MyCommandClass);
        
        ...
   
        //now it can be executed using 'executeCommand'
        var event:Event = new Event(Event.EVENT);
        controller.executeCommand(event);
       }
      }
     }

mvcEventDispatcherproperty 
protected static var mvcEventDispatcher:MVCEventDispatcher

The MVCEventDispatcher instance.

This is normally used in order to notify the Controller that it needs to execute an ICommand.

See also

org.jsmith.mvc.events.MVCEvent
org.jsmith.mvc.events.MVCEventDispatcher

Example
To dispatch an event via the MVCEventDispatcher, call its dispatchEvent method: 
   
     package
     {
      import org.jsmith.mvc.events.MVCEvent;
      ...
   
      public class MyClassView extends View implements IView
      {
       public function MyClassView()
       {
        myButton.addEventListener(MouseEvent.CLICK, onClick);
        ...
       }
   
       public function onClick(e:MouseEvent):void
       {
        var event:MVCEvent = new MyMVCEvent(MyMVCEvent.EVENT);
        mvcEventDispatcher.dispatchEvent(event);
       }
      }
     }
