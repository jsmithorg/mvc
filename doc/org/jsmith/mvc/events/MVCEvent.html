Classpublic class MVCEvent
InheritanceMVCEvent flash.events.Event

The MVCEvent is an event object for use with the MVCEventDispatcher in our Model-View-Controller framework.

MVCEvent objects are the transport and communication mechanism.

Parameters
type:String — A string value representing the event type.
 
data:Boolean (default = null) — An arbitrary object for passing data arguments along with the event.
 
bubbles:Boolean (default = false) — Whether or not the event will bubble.
 
cancelable:* (default = false) — Whether or not the even is cancelable.
